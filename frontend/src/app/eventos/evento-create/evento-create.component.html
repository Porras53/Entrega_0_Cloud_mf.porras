Crear evento

Debe ingresar un nombre para el evento
El nombre debe tener por lo menos un carácter
El nombre no debe exceder los 50 carácteres
Debe ingresar un lugar para el evento
El lugar debe tener por lo menos un carácter
El lugar no debe exceder los 50 carácteres
Debe ingresar una FECHA INICIAL para el evento
LA FECHA INICIAL debe tener por lo menos un carácter
LA FECHA INICIAL no debe exceder los 50 carácteres
Debe ingresar una FECHA Final para el evento
LA FECHA Final debe tener por lo menos un carácter
LA FECHA Final no debe exceder los 50 carácteres
Debe seleccionar la categoria del evento
Debe ingresar una direccion para el evento
La direccion del evento debe tener por lo menos un carácter
La direccion del evento no debe superar los 50 carácteres
Debe seleccionar el modalidad del evento
